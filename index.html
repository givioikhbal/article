Cara Membuat dan Mengatur Administrator Profile pada FortiGate Firewall

Ditulis oleh Givio Ikhbaludhuda | 21 Juli 2025

Secara default, FortiGate memiliki akun administrator dengan nama pengguna admin dan tanpa kata sandi. Untuk mencegah akses tidak sah ke FortiGate, akun ini harus dilindungi dengan kata sandi. Administrator tambahan dapat ditambahkan untuk berbagai fungsi, masing-masing dengan nama pengguna, kata sandi, dan serangkaian hak akses yang unik.

Administrator profiles

Administrator Profile menentukan apa yang dapat dilakukan administrator saat masuk ke FortiGate. Saat Anda membuat akun administrator, Anda juga menetapkan profil administrator yang menentukan apa yang dapat dilihat administrator. Tergantung pada jenis pekerjaan, tingkat akses, atau senioritas administrator, Anda dapat mengizinkan mereka untuk melihat dan mengonfigurasi sebanyak atau sesedikit yang diperlukan.

Secara default, FortiGate memiliki akun administrator admin yang menggunakan profil super_admin. Berikut jenis-jenis profile yang ada pada FortiGate:

Profile Description
super_admin Profil super_admin menawarkan akses read dan write penuh ke semua komponen FortiOS, termasuk kemampuan untuk mengelola administrator lain dan melakukan fungsi penting seperti mencadangkan dan memulihkan konfigurasi.
prof_admin Profil prof_admin, meskipun menyediakan akses administratif yang luas, secara umum tidak mencakup kemampuan untuk mengatur ulang kata sandi untuk administrator lain atau, dalam versi FortiOS yang lebih baru, mungkin tidak mengizinkan perintah diagnosis secara default.
read_user Profil read_user (atau read-only) hanya memberikan akses lihat saja ke konfigurasi dan status FortiGate, dan tidak dapat membuat perubahan apa pun.

Penjelasan Detail Profil:

super_admin:

  • Memberikan akses penuh ke semua fitur, pengaturan, dan fungsi FortiGate.
  • Dapat mengelola semua akun administrator, termasuk membuat, menghapus, dan memodifikasinya.
  • Penting untuk tugas-tugas penting seperti backup dan restore.
  • Installation & Configuration of Huawei Network Devices
  • Akun administrator default pada FortiGate baru menggunakan profil ini.

prof_admin:

  • Menawarkan kemampuan administratif yang signifikan, tetapi dengan beberapa keterbatasan dibandingkan dengan super_admin.
  • Perbedaan utama yang sering disebutkan adalah ketidakmampuan untuk mengatur ulang kata sandi administrator lain.
  • Pada versi FortiOS terbaru (misalnya, 7.4.8 dan yang lebih baru), perintah diagnosis mungkin dinonaktifkan secara default untuk prof_admin.
  • Cocok untuk peran administratif yang memerlukan akses luas tetapi tidak memerlukan kontrol penuh atas akun admin lain atau tools diagnostik tingkat lanjut tertentu.

read_user:

  • Menyediakan akses read-only ke antarmuka FortiGate dan pengaturannya.
  • Pengguna dengan profil ini dapat melihat konfigurasi, log, dan status, tetapi tidak dapat melakukan perubahan atau modifikasi apa pun pada sistem.
  • Ideal untuk peran pemantauan, audit, atau pemecahan masalah terbatas.
  • Profil ini dapat dibuat dan dikustomisasi untuk memberikan akses ke area tertentu di FortiGate.

Membuat Profil secara Custom

Untuk membuat profil di GUI:

  1. Buka System > Admin Profiles.
  2. Pilih Create New.
  3. Konfigurasikan pengaturan berikut:
    • Name.
    • Access permissions.
    • Override idle timeout.
  4. Pilih OK.

Tambahkan Local Administrator

Secara default, FortiGate memiliki satu super admin bernama admin. Anda dapat membuat lebih banyak akun administrator dengan hak istimewa yang berbeda.

Untuk membuat profil di GUI:

  1. Buka System > Administrators.
  2. Pilih Create New > Administrator.
  3. Tentukan Username.

    4. Jangan gunakan karakter < > ( ) # " ' pada nama pengguna administrator.

    Penggunaan karakter ini pada nama pengguna administrator dapat menimbulkan kerentanan cross-site scripting (XSS).

  4. Tetapkan Type ke Local User.
  5. Tetapkan kata sandi dan bidang lainnya.
  6. Pilih OK.

Setelah melakukan langkah-langkah di atas, cobalah untuk logout dan login menggunakan akun administrator yang telah dibuat.
